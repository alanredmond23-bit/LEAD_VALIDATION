# Lead Validation Audit Tools

This folder contains the assets and instructions for the "Anthony Lead Forensics" workflow to validate a vendor lead file (pet/dental "free scan/review") and generate evidence suitable for publisher disputes or refunds.

## Overview

The Anthony Lead Forensics system helps validate vendor lead quality by:
- Analyzing lead files for authenticity and completeness
- Generating evidence packages for disputes
- Creating audit trails for refund requests
- Providing forensic analysis of lead generation practices

## Workflow Steps

1. **Lead File Analysis**
   - Import vendor lead file (CSV/Excel format)
   - Validate contact information completeness
   - Check for duplicate entries
   - Analyze lead source attribution

2. **Quality Assessment**
   - Contact validation (phone/email verification)
   - Geographic analysis
   - Timing pattern analysis
   - Lead scoring based on completeness

3. **Evidence Generation**
   - Create audit report with findings
   - Generate charts and visualizations
   - Document quality issues
   - Prepare dispute evidence package

4. **Documentation**
   - Save audit results
   - Create evidence files
   - Generate summary reports
   - Archive for future reference

## Output Formats

The system generates several output formats:
- PDF audit reports
- Excel analysis spreadsheets
- CSV data exports
- Chart/graph images
